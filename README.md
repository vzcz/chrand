## Usage/Examples
```
    >Help
```    
 Random selection  
```
    >random <select> <select> <select> <select>
```   
Example: 
```
    >random ali ahmad saad khalid
```
Gives you a set of photos (the number of photos is optional)
```
    >image <number of images> <image name>
```
Example: 
```
    >image 2 sky
```
Generate free proxy for different types
```
    >proxy <type>
```
types:  
[1]http  
[2]socks4  
[3]socks5  

Example: 
```    
    >proxy http
```
